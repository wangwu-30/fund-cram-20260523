# 基金从业资格考前冲刺网页

本地启动：

```bash
python3 -m http.server 4173
```

打开：<http://127.0.0.1:4173>

内容范围：

- 知识框架：按科目、章节、概念卡自顶向下组织。
- 脑图 + 图谱：顶层脑图看章节，局部图谱看先修、易混、同题考查和错题弱点。
- 原创题库：401 道单选练习题，覆盖 79 个核心概念。
- 题解追问：每道题答完后可继续问选项、概念、公式和易错点。
- 进度记录：错题、收藏、弱概念和模拟记录保存在本机浏览器 localStorage。
- 扩展题库：支持导入合规来源的 JSON 题目。

说明：

- 题库为原创练习题，不是官方真题，也未复制商业题库。
- 依据中国证券投资基金业协会 2026 年 5 月考试公告和 2025 年度修订大纲整理。
